# EEZ_SCHIZOPHRENIA
Hybrid ML + CNN model for detecting schizophrenia using EEG signals and clinical features. Includes preprocessing, feature extraction, and model comparison.


# 🧠 EEG-Based Schizophrenia Detection

## 📌 Project Overview
This project focuses on detecting schizophrenia using EEG (Electroencephalogram) brain signal data. Machine Learning and Deep Learning models are used to classify subjects as schizophrenia patients or healthy individuals.

The system combines EEG signal features with clinical data such as age and gender to improve prediction accuracy.

---

## 🎯 Objectives
- Analyze EEG signals to detect schizophrenia patterns
- Extract meaningful statistical features from EEG data
- Apply machine learning and deep learning models
- Compare model performance and identify the best model

---
